# **App Name**: Splitz Team Generator

## Core Features:

- File Upload: Allow users to upload Excel or CSV files containing a list of names or rows.
- Team Configuration: Enable users to specify the number of teams and optionally provide custom team names.
- Data Processing: Read and process the uploaded file, shuffle the rows randomly, and split them into the specified number of teams.
- Team Assignment: Ensure teams are of mostly equal size (maximum size difference of 1) and add 'TeamName' and 'TeamIndex' columns to the data.
- File Generation: Generate one XLSX file per team, a combined XLSX file with each team on a separate sheet, and a ZIP archive containing all team files.
- Download Links: Display download links on the results page for each generated team file, the combined XLSX, and the ZIP archive.

## Style Guidelines:

- Primary color: Black (#000000) to represent organization and structure.
- Accent 1 color: Electric Blue (#1E6AFF) for download links and actionable items, to add contrast and visual interest.
- Accent 2 color: Gold Amber (#FFC741).
- Accent 3 color: Magenta Purple (#C141FF).
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern, neutral feel.
- Simple, clean layout with a focus on ease of use. Logo prominently displayed on the upload page.
- Use clear, recognizable icons for file uploads and downloads.
